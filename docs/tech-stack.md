# Tech Stack

| Layer | Technology | Purpose |
| :--- | :--- | :--- |
| **Frontend Framework** | Next.js (App Router) | High-performance React framework for Web & Admin. |
| **Mobile Framework** | Expo / React Native | Cross-platform native application development. |
| **Styling** | TailwindCSS / NativeWind | Utility-first CSS for consistent design across Web & Mobile. |
| **State Management** | Zustand | Lightweight state management with persistence middleware. |
| **Database** | PostgreSQL (Supabase) | Scalable relational database for products, orders, and users. |
| **Authentication** | Supabase Auth | Secure user and administrative authentication. |
| **File Storage** | Supabase Storage | Cloud storage for product images and media assets. |
| **AI Model** | Google Gemini AI | Powering the Brand Manager and content generation. |
| **ORM** | Drizzle ORM | Type-safe database interactions for the mobile app. |
| **Deployment** | Vercel | Automated CI/CD for web components. |
| **Tools** | GitHub Actions, Prettier, ESLint | Maintaining code quality and automation. |
